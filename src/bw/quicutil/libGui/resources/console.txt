** Console **


1 - Using the Console

Press TAB to open/close the console.  When the console is open, type in a command and press the enter key.
For more information enter "help" into the command line.


2 - Changing a value/key

To set a bool or value option, use the command "set <option> <value>".
If you want to see what the option's current value is, type in the name of the option and hit enter.
For a list of options, enter in "show options".

To set a key option, use the command "bind <key> <action>".
If you want to see what the option's current value is, type in the action and hit enter.
For a list of actions, enter in "show actions".
