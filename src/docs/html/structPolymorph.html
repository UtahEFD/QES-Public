Detailed Description

template<typename generic, typename thisType>
struct Polymorph< generic, thisType >

This structure is for grouping information necessary for parsing out polymorphic objects from an XML. This takes two template types, one for the base class, and one for the inheriting class. The constructor takes in a string which is used as a tag for finding values in the XML
